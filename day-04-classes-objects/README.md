\# Day 4 - Classes, Objects, Constructor, Getter and Setter



Today, I learned the basics of object-oriented programming in Java.



\## Topics Covered



\- Class

\- Object

\- Field

\- Constructor

\- Private fields

\- Getter methods

\- Setter methods

\- Methods

\- this keyword

\- Encapsulation

\- if-else inside methods



\## Description



I created a Student class with private fields such as name, age, and grade. I used a constructor to create student objects. I practiced getter methods to read private fields and setter methods to update private fields safely.



I also used methods to display student information, check pass/fail status, and show letter grades.
